# Changelog

## [1.0.0] - 2026-02-20

### Hinzugefügt
- Initiale Version der Universal Media Card
- Multi-Source-Support für SkyQ, Android TV, Spotify und Generic Media Player
- App-Icon-Anzeige für aktive Quelle
- Source-Selector zum Wechseln zwischen Quellen
- Media Controls (Play/Pause, Previous/Next, Volume)
- Individuelle Audio-Quellen-Konfiguration
- Kompakte Ansicht
- Visueller Editor für Konfiguration
- HACS-Unterstützung

### Features
- Unterstützt mehrere Medienquellen gleichzeitig
- Nahtloser Wechsel zwischen Quellen ohne neue Card
- Zeigt App-Icon/Logo der aktiven Quelle
- Einheitliche Steuerung für alle Quellen
- Konfigurierbare Audio-Ausgabe pro Quelle
